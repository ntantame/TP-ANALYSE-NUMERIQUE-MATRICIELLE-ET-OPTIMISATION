 TP – Analyse Numérique Matricielle et Optimisation

Description
Ce projet s’inscrit dans le cadre du cours **Analyse Numérique Matricielle et Optimisation**.
Il vise à implémenter et comparer des méthodes itératives de résolution des systèmes linéaires.

 Objectifs
- Implémenter les méthodes de **Jacobi** et **Gauss-Seidel**
- Étudier la convergence des méthodes
- Comparer la vitesse de convergence
- Appliquer les méthodes sur des matrices tests

 Outils utilisés
- Langage : Python
- Bibliothèques : 
- Environnement : VS Code et jupyter
